# MyVim  
# 我的vim配置文件,持续更新  
# 使用方法：
## clone下我的项目，把vimrc移动到~/下，修改vimrc为.vimrc 
> ***sudo git clone https://github.com/omega-Lee/MyVim***  
> ***sudo cp vimrc ~/***  
> ***sudo mv vimrc .vimrc***     


# 特殊键位说明
>
    **S :强制保存 **  
    **Q :强制退出 **  
    **Alt:ESC **
